## Command: ifconfig
Purpose:Displays network interface configuration information.
Output Summary:Shows interface names, IP addresses, MAC addresses, and packet statistics.
Use Cases:Used to check IP configuration and troubleshoot interface-related problems.
My Observations:The en0 interface shows the assigned IP address used for internet connectivity.
